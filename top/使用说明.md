## 🎯 如何使用

### 方法 1：直接在浏览器中打开（最简单）

```bash
# 在文件管理器中双击打开
index-complete.html
```

### 方法 2：使用本地服务器

```bash
# 进入目录
cd kissapi-tutorial

# 使用 Python 启动服务器
python -m http.server 8000

# 或使用 Node.js
npx http-server

# 然后在浏览器访问
http://localhost:8000/index-complete.html
```

### 方法 3：部署到 GitHub Pages

```bash
cd kissapi-tutorial

# 初始化 Git 仓库
git init
git add .
git commit -m "Add KissAPI tutorial"

# 连接到你的 GitHub 仓库
git remote add origin https://github.com/你的用户名/你的仓库名.git
git branch -M main
git push -u origin main
```

然后在 GitHub 仓库设置中启用 GitHub Pages：
- Settings → Pages
- Source: Deploy from a branch
- Branch: main / (root)
- Save

几分钟后访问：`https://你的用户名.github.io/你的仓库名/index-complete.html`
